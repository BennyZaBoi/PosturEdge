# Back-Posture-Edge-Computing-Device

This is a back posture correction device.

The tools that we used includes the Arduino IDE, Google Firebase for Cloud Connection, and parts listed in the documentation.

Some changes in the code is the fact that we removed the password, API Key, and Website in the code for protective reasons. 


## Setup

1. Copy `secrets_example.h`
2. Rename it to `secrets.h`
3. Fill in your own:
   - WiFi SSID
   - WiFi password
   - Firebase API key
   - Firebase Realtime Database URL

*** use latest version of upper and lower back code, use file named 
finalDraftStableAngel_upperBack and finalDraftStableAngel (lower back)
